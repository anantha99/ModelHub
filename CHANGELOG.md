# Changelog

## 0.1.1 - 2026-07-04

First installable Windows preview release.

- Enables a Tauri NSIS current-user installer for Windows 10/11 x64.
- Adds release metadata, MIT licensing, and installer-oriented documentation.
- Hides the extra console window for release builds.
- Adds a repeatable self-signed Authenticode signing script for preview artifacts; Windows trust warnings may still appear.
- Generates local installer, public certificate, and release notes assets for GitHub Release upload.

## 0.1.0 - 2026-07-04

Initial Windows MVP source release.

- Adds the Tauri desktop shell and tray support.
- Adds Local, Explore, Downloads, Runtimes, and Settings pages.
- Adds Hugging Face cache scanning, LM Studio folder scanning, and Ollama local API scanning.
- Adds Hugging Face public search, model details, staged downloads, and Hugging Face cache installation.
- Adds safe local delete boundaries, Explorer open actions, and runtime/system information basics.
